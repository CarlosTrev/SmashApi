Esta aplicacion muestra y recibe datos que recibe a traves de una Api de Super Smash Bros Ultimate

https://smashbrosapi.com/api/v1/ultimate/characters <--- Este es el link que hace la solicitud con la que trabaja el proyecto

Para que funcione, solamente basta con clonarlo, si presenta alguna falla asegurese de que el build.glade.kts (app) esta sincronizado con el proyecto,
y si en el splashscreen no sale nada, compruebe que la imagen "ssbu.png" esta descargada en la carpeta "drawable".
Por ultimo, es necesario que el equipo donde se ejecute la app este conectado a internet, caso contrario se ejectura, pero no mostrara nada.
